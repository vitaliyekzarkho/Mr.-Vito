# GitHub Upload Guide

## 1. Create a GitHub repository

Recommended repository name:

```text
ireland-homelessness-analytics-capstone
```

Suggested description:

```text
End-to-end analytics capstone project on homelessness in Ireland using public datasets, SQL and Power BI-ready reporting artifacts.
```

## 2. Initialise and commit locally

From the project folder:

```powershell
git init
git add .
git commit -m "Initial capstone project upload"
```

## 3. Connect to GitHub

Replace `YOUR_USERNAME` with your GitHub username:

```powershell
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/ireland-homelessness-analytics-capstone.git
git push -u origin main
```

## 4. Recommended GitHub settings

- Keep the repository public if it is intended for a portfolio.
- Add topics such as:

```text
data-analysis
sql
power-bi
analytics
ireland
homelessness
capstone-project
portfolio
```

## 5. What to show in portfolio

Use the README as the landing page and highlight:

- Executive summary
- Final presentation
- SQL views
- Power BI page mockups
- Data notes and limitations
